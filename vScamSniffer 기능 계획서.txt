<챗봇 기반 보이스피싱 예방 서비스>

 - 보이스피싱 판별 프롬프트

📌 사용자가 통화 내용을 음성 파일로 입력
📌 입력된 음성 파일을 STT를 통해 text로 전환
📌 판별 결과: 보이스피싱 가능성이 높음 (위험도: 92%)
📌 근거: 
 - '계좌 정지'와 같은 긴급성을 강조하는 표현이 포함됨
 - 사용자의 클릭을 유도하는 URL 포함됨
📌 조치 권장:
 - 공식 금융기관에 직접 연락하여 확인
 - 링크를 클릭하지 말고 금융감독원(1332)에 신고

 - 보이스피싱 롤플레잉

📌 보이스피싱 상황을 사용자가 선택 or 랜덤으로 결정
📌 STT 활용하여 사용자가 답변
📌 올바른 대처인지 판단하여 평가 or 올바른 답변 제시

 - 보이스피싱에 이미 당했을 때 대처방안 제시

📌 사용자에게 보기 제공
 - 1. 신분증을 촬영하여 상대에게 전송하였나요?
 - 2. 보유 중인 계좌의 계좌번호나 카드 번호를 상대에게 알려주었나요?
 - 3. 개인정보가 상대에게 유출되었나요?
 - 4. 휴대폰에 해킹 어플 등을 설치하였나요?
📌 이런 식의 보기를 제공하고 사용자가 각 보기에 해당하면 체크 혹은 번호를 골라 답변하면 대처방안 제시
